# Group 9 Library Management System

This is a Python project group 9 demonstrating SQLAlchemy models and Alembic migrations.

## Project overview

This repository defines basic models for a library (currently `Book` and `Author` in `library.py`) and includes an Alembic migrations folder. Use this project as a starting point to add CRUD operations, API layers, or tests.

## Quick start

Install and activate dependencies (using `pipenv`):

```
pipenv install
pipenv shell
```
```
python3 -m venv .venv
source .venv/bin/activate
```

Run a quick sanity check that creates the SQLite database and tables:

```
python library.py
```

## Alembic (migrations)

1. Ensure `alembic/env.py` imports the `Base` metadata from `library.py` (set `target_metadata = Base.metadata`).
2. Create an autogenerated migration:

```
alembic revision --autogenerate -m "Initial migration"
```

3. Apply migrations:

```
alembic upgrade head
```

## Project structure

- `library.py` - SQLAlchemy models for `Book` and `Author`
- `alembic.ini` - Alembic configuration.
- `migrations/` - existing Alembic migration scripts and env template.
- `Pipfile` - project dependencies (if using `pipenv`).
